delete-plugin - delete-poject
=============================

NAME
----
delete - Delete project

SYNOPSIS
--------
>     DELETE /projects/{project-name}/delete-project-plugin~delete-project

DESCRIPTION
-----------
Delete project

OPTIONS
-------

--preserve
> Don't remove GIT repository directory.

--force
> Delete project even if it has open changes.

ACCESS
------
Caller must be a member of a group that is granted the 'Delete Project'
capability (provided by this plugin) or be a member of the Administrators
group.

EXAMPLES
--------

Delete project 'foo' including GIT repository directory

>     curl -X DELETE --digest --user joe:secret \
>       http://host:port/a/projects/foo/delete-project-plugin~delete-project

Delete project 'foo' even if it has open changes, but preserve GIT repository
directory

>     curl -X DELETE -H "Content-Type: application/json" \
>       -d '{force: true, preserve: true}' \
>       --digest --user joe:secret \
>       http://host:port/a/projects/foo/delete-project-plugin~delete-project
